The World Wide Web

The web is possible due to your computer being connected online and using the hypertext transfer protocol (http) to translate text mainly html, from the server back to your computer.

HTML Coding Basics
-Note that I will use curly brackets { } to replace < >.

Tags / Elements -
 What is used to tell the stupid computer what to do and when to do it. When writing a line of code it is called an element. For instance when you start a paragraph tag{p}, write some text, then close the paragraph tag{/p}. That would be considered an element. This video explains it in more detail.

Void Tag -
 A void tag never needs to be closed. An example of a void tag would be the break line tag {br}.

Attributes -
 Add additonal value into a tag. {tag attribute="value"} Content{/tag}

Inline and Block Elements -
Inline elements will normally stay inline with other inline elements. Athough they sometimes can start with a new line, but will never end with a new line.

Block elements will start and end with a new line. Basically creating an invisible block around the element.

Computers are stupid -
 This I think is a really good to keep in mind when coding. You need to tell the computer exactly what you want it to do and in the order you want to do it. Also known as writting in the correct syntax.
